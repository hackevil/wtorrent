#Node Scaffold
Scaffolding to quickly start up a node app.

## Includes
1. Sequelize : database management
    - sqlite 3 : development
    - postgres : production
2. Express : easy server setup
3. Underscore : scripting utilities
4. Babel : transpiling ES6
5. Mocha : testing

##Running Development
Start the server
```$
    npm run start 
```
Begin transpiling ES6
```$
    npm run build 
```
